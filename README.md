# workflow-automation-platform
Enterprise Workflow Automation Platform built with Java, Spring Boot, Microservices, Kafka, Hibernate, PostgreSQL, Redis, Docker, and Spring Security.

## Features

- User Authentication & Authorization (JWT)
- Workflow Creation & Management
- Dynamic Workflow Execution Engine
- Event-Driven Communication using Kafka
- Notification Service
- Execution History & Audit Logs
- Analytics Dashboard
- RESTful APIs
- Dockerized Microservices
- API Gateway
- Redis Caching
- Swagger API Documentation

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Cloud
- Hibernate (JPA)
- Apache Kafka
- PostgreSQL
- Redis
- Docker
- Maven
- OpenAPI (Swagger)
